Bienvenido

Hola y bienvenido a este pequeño proyecto el cual te permitira tener un registro de creacion de tareas. Solo necesitas registrarte como usuario para ingresar. Tus datos quedaran registrados al igual que tus tareas.

Este proyecto es meramente con fines de aprendizaje y poder aplicar todos mis conocimientos en angular de la manera mas optima y utilizando las mejores practicas. Addicional a este proyecto se incorporaron tecnologias de backend con autenticación.
